# Local Cursor Agent VS Code Extension

Minimal UI-only extension.

## Run

1. Open `C:\Users\Enot.DESKTOP-C19QK7E\Desktop\LocalCursorAgent\vscode-extension` in VS Code.
2. Press `F5`.
3. Run command: `Local Cursor Agent: Open Panel`.

## What it does

- workspace picker
- task textbox
- send button
- logs
- result area

Backend agent connection is not wired yet.
